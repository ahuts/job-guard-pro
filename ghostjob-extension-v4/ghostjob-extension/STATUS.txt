## Key Features

✅ Injects "Check for Ghost Job" button on LinkedIn
✅ Extracts job data from page (title, company, location, description)
✅ Sends to GhostJob API for analysis
✅ Shows Ghost Score (0-100) with color coding
✅ Popup UI for extension status
✅ Service worker for background API calls

## API Integration

- Endpoint: https://jobghost-gamma.vercel.app/api/scrape-job
- Method: POST
- Body: { url, title, company, location, description }
- Response: { success: true, data: { ghostScore, ... } }
